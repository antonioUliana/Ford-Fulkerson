# Ford-Fulkerson
Implementação de algoritmo de Ford-Fulkerson em python para obter o fluxo máximo em redes.  Cria-se uma matriz para representar as capacidades (como se fossem pesos).
A passagem da matriz será assim:
graph = [[0, 16, 13, 0, 0, 0],
              [0, 0, 10, 12, 0, 0],
              [0, 4, 0, 0, 14, 0],
              [0, 0, 9, 0, 0, 20],
              [0, 0, 0, 7, 0, 4],
              [0, 0, 0, 0, 0, 0]]
